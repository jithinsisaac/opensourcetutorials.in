Linux alternatives to windows softwares

September 12, 2019 Linux

In this post we will list all the Linux softwares equaivalent to windows softwares

Microsoft Office (Office Suite)

  • Libre Office Suite
  • Open Office
  • AbiWord Download

Microsoft Outlook

  • Evolution
  • Thunderbird

Notepad / Wordpad (Text Editor)

  • gedit
  • atom Can be used as a text editor
  • vscode Open Source Editor from Microsoft
  • Kate

Internet Explorer (Web Browser)

  • Chromium
  • Firefox

Photoshop (Graphics Editor)

  • Gimp
  • Inkscape
  • Krita
  • MyPaint

Movie Maker (Video Editor)

  • OpenShot
  • Kdenlive
  • Pitivi

Windows Media Player (Video Viewer)

  • VLC
  • Mplayer
  • Parole Media Player

Adobe Reader (PDF Reader)

  • Adobe Reader (Its available for Linux also )
  • Foxit Reader
  • Okular

Windows Photo Viewer (Photo Manager)

  • Shotwell
  • gThumb

Messangers

  • Pidgin
  • Skype
  • Empathy

Nero Burning Rom (CD/DVD Burner)

  • Brasero
  • K3b

Softwares like teamviewer

  • anyDesk
  • RealVNC
  • Remmina

Torrent

  • Transmission

FTP clients
